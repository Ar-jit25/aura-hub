# 🌌 Aura Hub

**Aura Hub** is a state-of-the-art, "Panoramic Zen" productivity dashboard designed for deep focus and high-performance workflow. It replaces cluttered list managers with a monumental, immersive COMMAND CENTER that keeps your intentions clear and your environment serene.


## ✨ Features

- **🏛️ Panoramic Zen Layout**: A wide-screen, no-scroll interface that distributes your focus and management tools across a single immersive viewport.
- **🕒 Monumental Timer**: A gigantic digital digital clock with custom focus, short break, and long break modes to anchor your deep work sessions.
- **🛡️ Multi-Profile Sanctuary**: Secure authentication using **JWT (JSON Web Tokens)** and **Bcrypt** hashing. Each user (handle-based) gets their own private, persistent data vault.
- **📋 Intentions Tower**: A sidebar-driven task management system with layered subtasks, rich notes, and secondary scrolling to keep the dashboard stable.
- **🎹 Aura Ambience**: Integrated YouTube IFrame API audio engine with curated soundscapes (Rain, Forest, Lofi) and pulsing visual indicators.
- **🧠 Wisdom Ticker**: A bottom-anchored quote feed with automatic 60-second rotation and manual navigation to fuel your focus.
- **⚙️ "Unbreakable" Local Vault**: A built-in high-performance JSON database engine that ensures persistent, multi-user support with ZERO external dependencies or complicated setup.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vite, Vanilla CSS (Glassmorphism), HSL Design Tokens.
- **Backend**: Node.js, Express.
- **Security**: JWT (Identity Signatures), BcryptJS (Password Encryption).
- **Core Engine**: Aura Local Vault (Hierarchical JSON Persistence).
- **Audio API**: YouTube IFrame API.

## 🚀 Getting Started

### 1. Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation
Clone the repository and install the dependencies for both parts of the hub:

```bash
# Clone the repository
git clone https://github.com/your-username/aura-hub.git
cd aura-hub

# Setup Backend
cd server
npm install
npm start

# Setup Frontend (In a new terminal)
cd client
npm install
npm run dev
```

### 3. Usage
1. Open **Aura Hub** at `http://localhost:5175/`.
2. **Establish Aura**: Create your unique handle and a secure password.
3. **Unlock Hub**: Log in to access your private intentions.
4. **Focus**: Set your timer, select your vibe, and conquer your day.
